Responsive Events

This website focuses on Javascript elements with emphasis on reponsive events and functions. I have a total of 9 images, where the images are split up into 3 themes, each theme having 3 pictures. The pictures themes are 'DC Universe', 'Marvel Universe' and 'Pokemon universe'. Refreshing the page will randoimize the pictures, and the 'time until', which consist of a countdown timer will also cause a refresh. Additionally, there will be a manual refresh button, and a counter for the number of refreshes. Below the image randomizer, is a gallery of the images, this will use a carousel to display them, with the names of the images on top. Feel free to give it a try!
